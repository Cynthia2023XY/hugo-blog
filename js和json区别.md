#review/knowledge/code/js
1. json只支持双引号，不支持单引号
2. json不支持功能函数function
3. js是一种编程语言，而json是一种数据格式
4. js中的对象和数组等数据结构可以通过json来表示和交换
5. json可以在多种语言间进行传输和解析，而js只能被js解析
